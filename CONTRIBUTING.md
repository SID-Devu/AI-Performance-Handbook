# Contributing to AI Performance Handbook

Thank you for your interest in contributing!

## Types of Contributions

- **New chapters**: Deep technical content
- **Case studies**: Real-world optimization examples
- **Diagrams**: Visual explanations
- **Corrections**: Typos, technical accuracy

## Writing Style

- Clear, technical writing
- Include code examples
- Add diagrams where helpful
- Reference sources

## Structure

- Chapters go in `chapters/`
- Case studies in `case-studies/`
- Diagrams in `diagrams/`

## Pull Request Process

1. Fork repository
2. Create feature branch
3. Follow existing format
4. Submit PR
